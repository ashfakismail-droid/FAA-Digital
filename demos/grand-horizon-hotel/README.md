# Grand Horizon Hotel

A polished, production-ready luxury hotel landing page built with HTML, CSS, and vanilla JavaScript.

## Features
- Premium dark navy and gold visual design
- Responsive hero, about, rooms, amenities, gallery, testimonials, booking, and contact sections
- Sticky navigation and smooth scrolling
- Mobile menu, scroll reveal animations, gallery modal, back-to-top button, and booking validation
- Semantic HTML and modular CSS/JS structure

## Run locally
Open index.html in a browser, or serve the folder with any static server.
